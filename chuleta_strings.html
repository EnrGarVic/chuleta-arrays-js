Chuleta de Métodos de String en JavaScript

1. Métodos para manipular strings

Métodos de manipulación
Método Modifica string Estructura básica Casos de uso Qué devuelve
toUpperCase() No str.toUpperCase() Convertir todas las letras a mayúsculas. Un nuevo string en mayúsculas.
toLowerCase() No str.toLowerCase() Convertir todas las letras a minúsculas. Un nuevo string en minúsculas.
trim() No str.trim() Eliminar los espacios en blanco al inicio y al final del string. Un nuevo string sin espacios al inicio o final.
repeat(count) No str.repeat(3) Repetir el string la cantidad de veces especificada. Un nuevo string repetido.

2. Métodos para buscar y extraer contenido

Métodos de búsqueda y extracción
Método Modifica string Estructura básica Casos de uso Qué devuelve
charAt(index) No str.charAt(2) Obtener el carácter en la posición dada. Un string con el carácter (vacío si está fuera de rango).
includes(substring) No str.includes("word") Comprobar si el string contiene el substring dado. true si lo contiene, false si no.
indexOf(substring) No str.indexOf("word") Encontrar la primera posición de un substring. El índice encontrado o -1 si no lo encuentra.
lastIndexOf(substring) No str.lastIndexOf("word") Encontrar la última posición de un substring. El índice encontrado o -1 si no lo encuentra.
slice(start, end) No str.slice(2, 5) Extraer una porción del string entre start y end. Un nuevo string con la porción extraída.
substring(start, end) No str.substring(2, 5) Extraer una porción del string, intercambiando start y end si es necesario. Un nuevo string con la porción extraída.
substr(start, length) No str.substr(2, 3) Extraer una porción del string con una longitud dada (obsoleto). Un nuevo string con la porción extraída.

3. Métodos avanzados

Método Modifica string Estructura básica Casos de uso Qué devuelve
replace(search, replacement) No str.replace("word", "newWord") Reemplazar la primera ocurrencia del search. Un nuevo string con el reemplazo.
replaceAll(search, replacement) No str.replaceAll("word", "newWord") Reemplazar todas las ocurrencias del search. Un nuevo string con todos los reemplazos.
split(separator) No str.split(",") Dividir un string en un array según el separador dado. Un array con las partes del string.
match(regex) No str.match(/regex/) Obtener las coincidencias de una expresión regular. Un array con las coincidencias o null si no hay ninguna.
