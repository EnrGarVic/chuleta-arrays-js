Chuleta de Métodos de Array en JavaScript

1. Métodos para agregar/remover elementos

Métodos de agregar y remover
Método Modifica array Estructura básica Casos de uso
push(...items) Sí arr.push(item1, item2) Agregar elementos al final del array.
pop() Sí arr.pop() Eliminar el último elemento del array.
unshift(...items) Sí arr.unshift(item1, item2) Agregar elementos al inicio del array.
shift() Sí arr.shift() Eliminar el primer elemento del array.
splice(pos, deleteCount, ...items) Sí arr.splice(1, 2, "a", "b") Eliminar o insertar elementos en una posición específica.
slice(start, end) No arr.slice(1, 3) Crear un nuevo array copiando desde start hasta end (sin incluir).
concat(...items) No arr.concat(item1, item2) Crear un nuevo array combinando otros arrays o elementos.

2. Métodos para buscar entre elementos

Métodos de búsqueda
Método Modifica array Estructura básica Casos de uso
indexOf(item, pos) No arr.indexOf("value", 1) Encontrar la posición de un elemento.
lastIndexOf(item, pos) No arr.lastIndexOf("value") Encontrar la última posición de un elemento.
includes(value) No arr.includes("value") Comprobar si el array contiene un valor.
find(func) No arr.find(item => item.prop > 10) Devolver el primer elemento que cumpla una condición.
findIndex(func) No arr.findIndex(item => item.prop > 10) Devolver el índice del primer elemento que cumpla una condición.
filter(func) No arr.filter(item => item.prop > 10) Crear un array con elementos que cumplan una condición.

3. Métodos para iterar sobre elementos

Método Modifica array Estructura básica Casos de uso
forEach(func) No arr.forEach(item => console.log(item)) Ejecutar una función para cada elemento.

4. Métodos para transformar el array

Método Modifica array Estructura básica Casos de uso
map(func) No arr.map(item => item * 2) Transformar cada elemento del array y devuelve un array de la misma longitud
sort(func) Sí arr.sort((a, b) => a - b),array.sort( (a, b) => a.localeCompare(b) ) Ordenar los elementos.
reverse() Sí arr.reverse() Invertir el orden de los elementos.
reduce(func, initial) No arr.reduce((acc, item) => acc + item, 0) Reducir a un solo valor.
split/join No 'a,b'.split(',') de string a array / arr.join(',') de array a string Convertir entre arrays y cadenas.

5. Métodos adicionales

Método Modifica array Estructura básica Casos de uso
Array.isArray(value) No Array.isArray(arr) Verificar si un valor es un array.
some(func) No arr.some(item => item > 10) Comprobar si algún elemento cumple una condición.
every(func) No arr.every(item => item > 10) Comprobar si todos los elementos cumplen una condición.
